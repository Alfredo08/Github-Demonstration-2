
<h2>
    Commands to setup github the first time
</h2>
<ol>
    <li>
        git init 
    </li>
    <li>
        git add .
    </li>
    <li>
        git commit -m "Message here"
    </li>
    <li>
        git remote add origin url-to-github
    </li>
    <li>
        git push -u origin main
    </li>
</ol> 
<h2>
    Extra commands to update github locally
</h2>
<ul>
    <li>
        git clone url-of-github
    </li>
    <li>
        git pull
    </li>
    <li>
        git status
    </li>
</ul>
